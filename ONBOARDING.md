# ONBOARDING (READ FIRST)

Repo is the only source of truth. No relying on chat memory.

## Goal
Write Telegram product posts (RU) using our template and only approved sources.

## Start order (mandatory)
1) `telegram_training_dump/02_rules/templates.md`
2) `telegram_training_dump/02_rules/voice.md`
3) `telegram_training_dump/02_rules/do.md` + `dont.md`
4) `telegram_training_dump/04_checklists/source_checklist.md`
5) `telegram_training_dump/03_tests/test_01_posts.md`

## Definition of Done (per post)
A post is DONE only if:
- It references a source link from `00_sources/channels.md`
- Product facts are not invented (only from source)
- Draft saved to `outputs_telegram/drafts/YYYY-MM-DD_<slug>.md`
- Optional: logged in `outputs_telegram/daily_digest/YYYY-MM-DD.md`

## Where to write
- Drafts: `outputs_telegram/drafts/`
- Published (after posting): `outputs_telegram/published/`
